## What is this?
This app aims to help users track their subscriptions and recurring bills. It also lets users connect with each other so they can set up a subscription sharing plan.

## Plans
- Send email to users when nearing a subscription deadline
- Add a page that visualizes the subscription cycles in a graph or calendar format
- Move data persistance from local storage to a database (requires setting up a backend server)
- Add a chat feature to connect users with each other so they can discuss how to split costs for subscriptions
